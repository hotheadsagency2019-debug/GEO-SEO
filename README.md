# GEO-SEO · AI Content Pipeline

Конвейер из 9 специализированных AI-агентов для автоматического написания SEO-статей.
Каждый агент выполняет одну функцию и передаёт результат следующему.

---

## Архитектура

```
INPUT (CSV / CLI)
      ↓
┌─────────────────────────────────┐
│  Agent 1: Keyword Analyzer      │  → структура статьи (H1, H2, ключи)
└───────────────┬─────────────────┘
                ↓
┌─────────────────────────────────┐
│  Agent 2: LSI & Semantic        │  → LSI-фразы, сущности, вопросы PAA
└───────────────┬─────────────────┘
                ↓
┌─────────────────────────────────┐
│  Agent 3: Fact Collector        │  → факты, статистика, кейсы
└───────────────┬─────────────────┘
                ↓
┌─────────────────────────────────┐
│  Agent 4: Article Writer        │  → черновик статьи (Markdown)
└───────────────┬─────────────────┘
                ↓
┌─────────────────────────────────┐
│  Agent 5: SEO Editor            │  → редактура: FAQ, таблицы, списки
└───────────────┬─────────────────┘
                ↓
┌─────────────────────────────────┐
│  Agent 6: HTML Formatter        │  → семантический HTML
└───────────────┬─────────────────┘
                ↓
┌─────────────────────────────────┐
│  Agent 7: Internal Link Analyzer│  → рекомендации перелинковки
└───────────────┬─────────────────┘
                ↓
┌─────────────────────────────────┐
│  Agent 8: Link Inserter         │  → HTML со вставленными ссылками
└───────────────┬─────────────────┘
                ↓
┌─────────────────────────────────┐
│  Agent 9: Final QA              │  → чеклист + финальный HTML
└───────────────┬─────────────────┘
                ↓
OUTPUT: publication-ready HTML + JSON artefacts
```

## Установка

```bash
pip install -r requirements.txt
```

Скопируйте `.env.example` → `.env` и вставьте свой Anthropic API key:

```bash
cp .env.example .env
# отредактируйте .env
```

## Использование

### Обработать CSV-файл

```bash
python main.py run --csv keywords_example.csv
```

С кастомной картой сайта для перелинковки:

```bash
python main.py run --csv keywords_example.csv --sitemap sitemap_example.json
```

Обработать только одну строку (по индексу):

```bash
python main.py run --csv keywords_example.csv --row 0
```

### Обработать одно ключевое слово

```bash
python main.py single \
  --keyword "нейросети для контекстной рекламы" \
  --cluster "AI для Яндекс Директ, ChatGPT для рекламы" \
  --intent "информационный" \
  --page-type "статья"
```

## Формат входных данных (CSV)

| Колонка            | Описание                                    | Пример                              |
|--------------------|---------------------------------------------|-------------------------------------|
| `main_keyword`     | Главный целевой запрос                      | нейросети для контекстной рекламы   |
| `cluster_keywords` | Кластер связанных запросов (через запятую)  | AI для Яндекс Директ, ChatGPT       |
| `search_intent`    | Тип интента                                 | информационный / коммерческий       |
| `page_type`        | Тип страницы                                | статья / лендинг / категория        |

## Формат карты сайта (JSON)

```json
[
  {"url": "/blog/seo-basics/", "title": "Основы SEO: полное руководство"},
  {"url": "/blog/yandex-direct/", "title": "Яндекс Директ: настройка и оптимизация"}
]
```

Если файл не указан, используется демо-набор из 10 страниц.

## Описание агентов

| # | Агент                    | Входные данные          | Выходные данные                                 |
|---|--------------------------|-------------------------|-------------------------------------------------|
| 1 | **Keyword Analyzer**     | KeywordRow              | title, H1, H2 с ключами, meta description       |
| 2 | **LSI Expansion**        | KeywordAnalysis         | LSI-фразы, сущности, PAA-вопросы                |
| 3 | **Fact Collector**       | KA + LSI                | факты, статистика, кейсы, источники             |
| 4 | **Article Writer**       | KA + LSI + Facts        | черновик в Markdown (SEO или GEO-структура)     |
| 5 | **SEO Editor**           | Draft + keywords        | редактура: FAQ 5+, таблица, цитата, матрица     |
| 6 | **HTML Formatter**       | Edited article          | HTML-контент с HotHeads классами (CSS — автоматически) |
| 7 | **Internal Link Analyzer**| HTML + sitemap         | список внутренних ссылок с релевантностью       |
| 8 | **Link Inserter**        | HTML + links            | HTML со вставленными ссылками                   |
| 9 | **Final QA**             | Linked HTML             | 9-пунктный чеклист HotHeads Band + финальный HTML |

## Режимы статей: SEO vs GEO

Поле `article_type` в CSV управляет режимом работы:

| `article_type` | Описание | Структура |
|---|---|---|
| `seo` | Стандартная SEO-статья для блога | H1 → лид → разделы H2 → кейсы → CTA |
| `geo` | GEO/AEO-статья для AI-цитирования | Quick Compare → Методология → Продукты → FAQ → Матрица решений |

### GEO-статья (для AI-цитирования)
Оптимизирована под ChatGPT, Perplexity, Google AI Overviews:
- Обязательная сравнительная таблица Quick Comparison
- Нумерованные разделы по каждому продукту (плюсы/минусы/когда выбрать)
- Матрица решений «Если вы… | Выберите… | Почему»
- Минимум 2 000 слов, аналитический нейтральный тон

## HTML-вёрстка: HotHeads Band Tilda Zero Block

Агент 6 генерирует **только HTML-контент** — без `<style>` тегов.
CSS инжектируется автоматически из `seo_pipeline/brand_config.py` (константа `BRAND_CSS`).

Это гарантирует, что стили **всегда точно соответствуют** бренду HotHeads Band, независимо от того, что генерирует модель.

### Фирменные цвета
| Назначение | Цвет |
|---|---|
| Основной тёмный | `#3e3e3e` |
| Светлый фон | `#e9e9e9` |
| Оранжевый акцент | `#e27829` |
| Тёмно-серый (CTA, шапки таблиц) | `#3C3C3C` |
| Граница | `#c8cdd9` |

### QA-чеклист (9 пунктов HotHeads Band)
1. Все H2/H3 содержат поисковый ключ
2. TOC синхронизирован с реальными H2
3. Callout только оранжевые и серые
4. Нет упоминаний Meta / Facebook / Instagram
5. Есть FAQ (5+), таблица сравнения, матрица решений
6. Есть цитата эксперта Алены Мумладзе
7. Есть CTA-блок со ссылкой на t.me/hotheads_band
8. Все три брейкпоинта (768/480/360px) присутствуют в CSS
9. Дата в article-meta — месяц и год (не просто год)

## Настройка карты сайта для перелинковки

```bash
python main.py run --csv keywords_example.csv --sitemap sitemap_example.json
```

JSON-файл с реальными страницами сайта:
```json
[
  {"url": "/blog/yandex-direct/", "title": "Яндекс Директ: гайд"},
  {"url": "/keisi/", "title": "Кейсы агентства HotHeads Band"}
]
```

**Модель:** по умолчанию `claude-opus-4-6`.
Для экономии — переключить на `claude-sonnet-4-6` в `seo_pipeline/agents.py` (константа `MODEL`).
